### ES6:

#### arrays:
- map():
```javascript
var numbers = [1, 4, 9];
var doubles = numbers.map( (num)=> {
  return num * 2;
});
```
- reduce():
```javascript
var numbers = [1, 4, 9];
var doubles = numbers.map( (num)=> {
  return num * 2;
});
```
